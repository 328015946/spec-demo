# PRD: 完善公司页面内容

## 背景
当前公司介绍页已具备基础结构，但内容信息密度与业务表达仍偏简略，需要增强可信度与可读性。

## 目标
- 丰富公司介绍页的业务内容与结构层次。
- 保持与现有站点风格一致，不引入复杂交互。
- 确保页面文案更适合对外展示。

## 范围
- 更新 `src/views/CompanyView.vue` 的页面内容与展示模块。
- 可新增静态模块：核心价值、服务领域、关键数据、联系引导等。

## 非目标
- 不改动后端接口。
- 不新增管理后台编辑能力。
- 不调整全站路由结构（除已有 `/company` 入口外）。

## 验收标准
- 页面至少包含：公司定位、服务能力、核心价值、发展数据、联系引导。
- 页面视觉风格与现有主题变量一致。
- lint 通过且无明显样式破坏。
