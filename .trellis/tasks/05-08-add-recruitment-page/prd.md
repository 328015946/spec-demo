# PRD: 新增招聘页面

## 目标
- 新增一个招聘页面用于展示岗位与招聘流程。
- 页面风格与现有站点保持一致。
- 在导航中提供招聘入口。
- 提升页面视觉层次与品牌表现，增强招聘信息的可读性和吸引力。

## 范围
- 新增 `src/views/RecruitmentView.vue`。
- 新增 `/recruitment` 路由。
- 在顶部导航增加“招聘信息”入口。
- 增强页面样式表现：主视觉渐变区、标签样式、岗位卡片 hover 动效、流程编号样式、CTA 强化。

## 验收标准
- 页面包含：招聘主视觉、岗位列表、福利亮点、投递流程、联系方式。
- 页面可通过导航和路由访问。
- lint 无新增报错。
- 招聘页面具备明显视觉层次（卡片、按钮、流程节点等），在桌面与移动端均保持可读性。
